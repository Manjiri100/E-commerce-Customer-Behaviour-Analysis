# E-commerce Customer Behaviour Analysis

## 📌 Overview
This project analyzes customer behaviour in an e-commerce dataset to identify purchasing patterns, segmentation, and insights using data science and machine learning techniques.

---

## 🎯 Objectives
- Understand customer purchasing behaviour
- Perform data cleaning and preprocessing
- Apply exploratory data analysis (EDA)
- Segment customers using clustering
- Generate business insights

---

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
